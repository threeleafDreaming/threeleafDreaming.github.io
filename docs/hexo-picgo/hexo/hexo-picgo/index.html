让你一篇文章走天下（SM.MS的妙用）

上篇解决了fluid的配置问题(可访问纯码农的博客)，现在就是关于图片的问题，如果想在多个平台发布文章，那么图片就不能存放在本地，那么就有了图床的概念。就是把你的图片上传到网上，我们获取到对应的url就可以一次编辑，多处使用了。

一、gitee无法使用外链

之前看网上可以结合Gitee,新建仓库然后使用picGo工具上传，鄙人试了一下，现在Gitee做了检查，不准使用图床外链，如果使用就会将仓库设为私有，这样就没法使用了，所以换了一种方式，使用SM.MS的方式

二、SM.MS的注册

1、使用sm.ms官网进行注册

2、登录后进入面板界面

3、获取token

4、Typora配置图床

在第5步的时候将文件配置为

5、复制图片即可返回文件的url地址，也可在官网中查看到已上传的图片

三、发布文章

后话

1、SM.MS一个账号只有5G的空间，所以如果存满了可以使用其他账号注册

2、图片可以访问，但是有些延迟，也可以使用腾讯云与阿里云等其他付费存储
