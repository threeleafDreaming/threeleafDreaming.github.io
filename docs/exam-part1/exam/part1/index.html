一、信息与信息化

信息

信息是物质，能量及其属性的标示的集合，是确定性的增加。
以物质介质为载体，传递和反映世界各种事物存在方式、运动状态的表征
信息不是物质，也不是能量

信息的定义：

香农-信息是用来消除随机不定性的东西。

信息系统

信息系统抽象模型：

管理模型、信息处理模型、系统实现条件

信息系统生命周期：

系统规划、系统分析（需求分析）、系统设计（概要设计、详细设计）、系统实施（编码、测试）、系统运行、系统维护

信息化

信息化内涵：

信息化的核心：

信息化体系

6要素：信息技术应用、信息技术和产业、信息化人才、信息化政策法规和标准规范（上鹰下鸡左人右龟）、信息资源、信息网络

信息化趋势

组织信息化趋势

工业互联网不是互联网在工业的简单应用
