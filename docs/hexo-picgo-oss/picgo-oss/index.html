花钱的图片服务就是好用哈（picGo+阿里云OSS）

接上一篇让你一篇文章走天下（SM.MS的妙用）,虽然SM.MS还是不错的，但是将文章发布到其他平台时，会出现获取不到图片的问题，如简书，公众号等，试了下，使用SM.MS是可以在知乎上正常使用的上篇文章。但为了保持各平台都可以通用，所以狠心买了OSS服务（心疼那几十块）。

一、注册阿里云账号并创建子账号赋权

可自行注册阿里云账号，也可以使用淘宝，钉钉登录。注册好阿里云账号可设置RAM子账号，一般使用子账号进行操作。

创建子账号后赋OSS权限，记住子账号的AccessKey ID、AccessKey Secret

二、购买OSS

1、进入到产品页面，进入OSS页面

2、购买OSS，选择购买方式

3、立即开通OSS

4、创建Bucket

创建bucket，创建后复制创建的名称与地域节点

三、配置PicGo中的OSS配置

图床配置

将上述的子账号的AccessKey ID、AccessKey Secret、创建的Bucket名称、地域节点（去掉.aliyuncs.com）后分别配置到设定KeyId、设定KeySecret、设定存储空间名、确认存储区域、指定存储路径。配置好设为默认图床

四、其他平台展示

1、知乎文章

2、csdn文章

3、头条文章
