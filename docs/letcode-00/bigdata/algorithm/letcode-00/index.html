一、定义

二分查找是一种算法，其输入是一个有序的元素列表。如果要查找的元素包含在列表中，二分查找返回其位置；否则返回null。

三、算法图解

如果想从1~24000个数字中查找到一个数字，需要多少步？
按照从1开始的话，那么可能需要24000次，但是使用二分查询最多需要$\log_2^{24000}$=18步。这就是二分查询的好处。
